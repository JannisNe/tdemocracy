Candidate selection
===================

The selection of nuclear alerts and the associated photometry is done by the `AMPEL broker`_. The plugin for the TDEmocracy project is `ampel-tdemocracy`. You can find a description of the selection criteria on their `GitHub`_.
.. _AMPEL broker: https://ampelproject.github.io/
.. _GitHub: https://github.com/JannisNe/ampel-tdemocracy
